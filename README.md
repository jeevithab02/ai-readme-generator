# 🚀 AI README Generator

An interactive web app that generates professional GitHub README files with live preview, copy, and download features.

## ✨ Features

* 🌐 Web-based UI (Flask)
* 👀 Live preview
* 📋 Copy to clipboard
* ⬇️ Download README.md
* 🎨 Theme support
* 📊 GitHub stats integration

## 🛠 Tech Stack

* Python
* Flask
* HTML/CSS/JS

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```
